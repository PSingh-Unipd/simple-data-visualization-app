# simple-data-visualization-app
Simple data visualization application build using Angular, Angular Material, NgRx and NGX Charts. 

### Installation
```
npm install
```
### Browse
```
ng serve
```
### Build Prod
```
npm run build --prod
```
### Demo
https://simple-data-visualization-app.surge.sh
